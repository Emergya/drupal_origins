 INTRODUCTION
 ------------
 Drupal Origins Extra modules extension (drupal_origins_extra) is a basic module which enable some extra drupal modules,
 and their dependencies. That modules has no a concrete focus, but they are very frequent at drupal sites.

 This module doesn't set any additional default settings during the install process.
 For more info about which modules are enabled with drupal_origins_extra, please see the section "Enabled modules" on
 this file.

 ENABLED MODULES
 ---------------
 As it was said at "Introduction" section, drupal origins extra extension enable several modules. That modules are very
 frequent at Drupal sites, such as Jcarrusel or breakpoints modules.
 The full list of modules enabled with Drupal Origins Extra are the following:

  * Flexslider (https://www.drupal.org/project/flexslider)
  * Jcarousel (https://www.drupal.org/project/jcarousel)
  * Colorbox (https://www.drupal.org/project/colorbox)
  * Breakpoints (https://www.drupal.org/project/breakpoints)
